# Basketball-Stats-CPP

C++ program that analyzes basketball team statistics using file input

Features: 
- Modular design using header and source files
- File input/output with structured data
- Clean console table formatting
- Team and league-wide stat analysis

How to Run:
1. Compile all source files:
   g++ main.cpp library.cpp -o stats
2. Run:
   ./stats
